## 🎯 使用说明

### 1️⃣ 打开应用

用浏览器打开 `index.html` 文件，即可看到首页。

1. 显示加载动画
2. 查询 Supabase 数据库
3. 显示结果或错误提示

1. 在商品列表页面
2. 点击任意"购买"按钮

5. 商品状态变为"已售"

## 📊 数据库结构

### user 表
```sql
CREATE TABLE user (
    user_id VARCHAR(50) PRIMARY KEY,
    user_name VARCHAR(100) NOT NULL,
    phone VARCHAR(20)
);
```

### item 表
```sql
CREATE TABLE item (
    item_id VARCHAR(50) PRIMARY KEY,
    item_name VARCHAR(200) NOT NULL,
    category VARCHAR(50) NOT NULL,
    price NUMERIC NOT NULL,
    status INT (0=未售, 1=已售),
    seller_id VARCHAR(50) REFERENCES user(user_id)
);
```

### orders 表
```sql
CREATE TABLE orders (
    order_id VARCHAR(50) PRIMARY KEY,
    item_id VARCHAR(50) REFERENCES item(item_id),
    buyer_id VARCHAR(50) REFERENCES user(user_id),
    order_date TIMESTAMP NOT NULL
);
```
